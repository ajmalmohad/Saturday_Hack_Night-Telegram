## Installation

Clone this Repo

```bash
  git clone https://github.com/arjun-ms/Saturday_Hack_Night-Telegram/tree/main/Code_Buddies
  cd Code_Buddies
```
Install Requirements

```bash
  pip install -r requirements.txt
```
Run

```bash
  python app.py
```
